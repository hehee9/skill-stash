# skill-stash

제가 실제로 쓰는 Codex 스킬 중 공유할 만한 것들을 모아 둔 저장소입니다.

향후 다른 스킬도 추가할 수 있습니다.

## 스킬

| 스킬 | 대상 | 설명 |
| --- | --- | --- |
| [code-design-guard](skills/code-design-guard/SKILL.md) | Codex 스킬 | 설계 의도에 맞춰 더 나은 구현 전 설계. 유지보수하기 좋은 코드, 누더기 코드 방지 |
| [code-review](skills/code-review/SKILL.md) | Codex 스킬 | 작업 후 구현 사항 검토 및 수정. 누락 방지, 유지보수성 향상 |

## 사용 방법

필요한 스킬 폴더를 [`skills/`](skills/)에서 골라 자신의 Codex 스킬 디렉터리에 복사해서 사용하면 됩니다.

## 참고

- 이 스킬들은 제 작업 방식과 취향을 반영합니다. 그대로 쓰기 전에 내용을 검토해 주세요.
